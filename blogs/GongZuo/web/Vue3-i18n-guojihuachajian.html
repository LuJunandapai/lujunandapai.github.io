Vue I18n 国际化插件

LuisApai2023/11/18WebVue3

# Vue I18n 国际化插件

国际化是指项目能够根据不同国家的语言进行转换，便于不同国家的用户使用。

Vue I18open in new window 是 Vue.js 的国际化插件，它可以轻松地将一些本地化功能集成到应用程序中

# I18n 安装

// 使用 npm
npm install vue-i18n@9
 
// 使用 yarn
yarn add vue-i18n@9

# main.js 文件中的完整代码

import {createApp} from 'vue'
import App from './App.vue'
import router from './router'
// Element Plus 引入
import ElementPlus from 'element-plus';
import "element-plus/dist/index.css";
import * as ElementPlusIconsVue from "@element-plus/icons-vue";
// 引入 axios
import axios from 'axios';
import ECharts from 'vue-echarts' // 引入ECharts
import "echarts";

// 1. 引入 i18n 国际化 语言包
import en from "@/langurage/en";
import zh from "@/langurage/zh";
import {createI18n} from "vue-i18n"; // 全局引入echarts

// 2. 组合语言包对象
const messages = {
    en,
    zh
}

// 3. 创建实例对象 在创建实例时，一定要将 legacy 属性设置为 false，因为 legacy 默认为 true，它支持使用 Vue2 框架中的 Options API，如果要在 Vue3 更好的使用 Composition API 模式，那么就需要将此属性手动设置为 false。
const i18n = createI18n({
    legacy: false,  // 设置为 false，启用 composition API 模式
    messages,
    locale: navigator.language // 获取浏览器的语言
    // locale: 'zh' // 默认显示中文
})

const app = createApp(App);
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
    app.component(key, component);
}
// 4. 挂载实例对象
app.use(i18n);
app.use(router);
app.use(ElementPlus);
app.provide('$axios',axios);
// 挂载ECharts   参数1:vue文件中挂载echarts的名字  参数2:挂载的组件
app.component('ECharts',ECharts);
app.mount("#app");

# I18n 使用

语言包是对象形式的，如果你定义的是简单的属性形式，要注意在组件中使用时的形式

// 标签赋值
{{ $t('navigateBar.recommend') }}

// 属性赋值
:content="$t('HomePageView')"
        
// 数据赋值
radioStatistics: this.$t('HomePageView.radio.my'),
        
// 方法里面使用 选项式
this.$t('navigateBar.recommend')

// 获取当前语言 中文_zh 英文_en
const currentLanguage = this.$i18n.locale;
console.log("当前语言: " + currentLanguage);

# 动态切换语言

想要切换语言时，修改 locale 属性就可以了

// 子组件调用父组件的方法 切换语言 值必须是语言文件的引入名称
switchover(command) {
  if (command == 'en') {
    this.$i18n.locale = 'en'; // 切换 I18n 语言包 选项式写法
    this.$emit('childClick', en);
  } else {
    this.$i18n.locale = 'zh'; // 切换 I18n 语言包 选项式写法
    this.$emit('childClick', zhCn);
  }
},
