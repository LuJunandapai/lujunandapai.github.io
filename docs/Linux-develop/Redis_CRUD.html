Redis API

LuisApai2023/05/19

# Redis 操作

# Redis 操作类

# RedisTemplate

  • 存入redis会造成乱码
  • 适用于 所有类型
@Autowired
private RedisTemplate redisTemplate;

# StringRedisTemplate

  • string 存入redis不会造成乱码
  • 只适用于 字符串 String
@Autowired
private static StringRedisTemplate stringRedisTemplate;

# 五大类型操作方法

# String 字符串 操作

  • 设置指定 key 的值为 value set(key, value)
redisTemplate.opsForValue().set("name", "Jack");
  • 获取指定 key 的值 get(key)
String name = redisTemplate.opsForValue().get("name");
  • 先获取指定 key 的值并返回，再将该 key 的值设为新值 value getAndSet(key, value)
String oldName = redisTemplate.opsForValue().getAndSet("name", "Rose");
  • 将指定 key 的值增加 delta，如果未提供 delta，则默认增加 1 increment(key, delta)
long count = redisTemplate.opsForValue().increment("count"); // 自增1
long count = redisTemplate.opsForValue().increment("count", 5); // 自增5
  • 将指定 key 的值减少 delta，如果未提供 delta，则默认减少 1 decrement(key, delta)
long count = redisTemplate.opsForValue().decrement("count"); // 自减1
long count = redisTemplate.opsForValue().decrement("count", 5); // 自减5
  • 获取指定 key 的值的长度，即字符串长度 size(key)
long length = redisTemplate.opsForValue().size("name");
  • 将指定的字符串追加到原有字符串的末尾 append(key, value)
int newLength = redisTemplate.opsForValue().append("name", " is a good boy.");
  • 仅在指定 key 不存在时，才将 key-value 存储到 Redis 中 setIfAbsent(key, value)
boolean success = redisTemplate.opsForValue().setIfAbsent("name", "Mike");
  • 批量设置多个 key-value multiSet(map)
Map<String, String> map = new HashMap<>();
map.put("name", "Tom");
map.put("age", "20");

redisTemplate.opsForValue().multiSet(map);
  • 批量获取多个 key 对应的 value multiGet(keys)
List<String> keys = Arrays.asList("name", "age", "address");
List<String> values = redisTemplate.opsForValue().multiGet(keys);

# Hash 哈希 操作

  • 这个方法可以将一个哈希表中的一个字段设置为指定值 opsForHash().put(key, hashKey, value)
redisTemplate.opsForHash().put("myHash", "field1", "value1");
  • 这个方法可以获取哈希表中指定字段的值 opsForHash().get(key, hashKey)
String value = redisTemplate.opsForHash().get("myHash", "field1").toString();
  • 这个方法可以将哈希表中指定字段的值增加指定的值（可以为负数）opsForHash().increment(key, hashKey, delta)
redisTemplate.opsForHash().increment("myHash", "field1", 10);
  • 这个方法可以判断哈希表中是否存在指定的字段 opsForHash().hasKey(key, hashKey)
Boolean exists = redisTemplate.opsForHash().hasKey("myHash", "field1");
  • 这个方法可以删除哈希表中指定的一个或多个字段 opsForHash().delete(key, hashKeys)
redisTemplate.opsForHash().delete("myHash", "field1", "field2");
  • 这个方法可以获取哈希表中所有字段和值的键值对 opsForHash().entries(key)
Map<Object, Object> entries = redisTemplate.opsForHash().entries("myHash");
  • 这个方法可以获取哈希表中所有字段的键值 opsForHash().keys(key)
Set<Object> keys = redisTemplate.opsForHash().keys("myHash");
  • 这个方法可以获取哈希表中所有字段的值 opsForHash().values(key)
List<Object> values = redisTemplate.opsForHash().values("myHash");

# List 列表 操作

  • 将一个或多个值插入到列表头部 leftPush(K key, V value)
ListOperations<String, String> ops = redisTemplate.opsForList();
ops.leftPush("mylist", "hello");
ops.leftPush("mylist", "world");
  • 将一个或多个值插入到列表尾部 rightPush(K key, V value)
ListOperations<String, String> ops = redisTemplate.opsForList();
ops.rightPush("mylist", "hello");
ops.rightPush("mylist", "world");
  • 移除并返回列表的左侧第一个元素 leftPop(K key)
ListOperations<String, String> ops = redisTemplate.opsForList();
String first = ops.leftPop("mylist");
System.out.println(first);
  • 移除并返回列表的右侧第一个元素 rightPop(K key)
ListOperations<String, String> ops = redisTemplate.opsForList();
String last = ops.rightPop("mylist");
System.out.println(last);
  • 获取列表指定范围内的元素 range(K key, long start, long end)
ListOperations<String, String> ops = redisTemplate.opsForList();
List<String> elements = ops.range("mylist", 0, -1);
System.out.println(elements);
  • 获取列表长度 size(K key)
ListOperations<String, String> ops = redisTemplate.opsForList();
Long size = ops.size("mylist");
System.out.println(size);
  • 保留列表指定范围内的元素，其它元素将被删除 trim(K key, long start, long end)
ListOperations<String, String> ops = redisTemplate.opsForList();
ops.trim("mylist", 0, 2);
  • 设置列表中指定位置的值 set(K key, long index, V value)
ListOperations<String, String> ops = redisTemplate.opsForList();
ops.set("mylist", 1, "redis");
  • 从列表中删除指定数量的值 remove(K key, long count, Object value)
ListOperations<String, String> ops = redisTemplate.opsForList();
ops.remove("mylist", 0, "hello");

# Set 集合 操作

  • 添加元素到集合中：opsForSet().add(key, values...)
// 添加单个元素到集合中
redisTemplate.opsForSet().add("myset", "value1");

// 添加多个元素到集合中
redisTemplate.opsForSet().add("myset", "value2", "value3");
  • 获取集合大小：opsForSet().size(key)
// 获取集合大小
Long size = redisTemplate.opsForSet().size("myset");
  • 判断集合中是否存在某个元素：opsForSet().isMember(key, value)
// 判断集合中是否存在指定元素
Boolean exists = redisTemplate.opsForSet().isMember("myset", "value1");
  • 获取集合中的所有元素：opsForSet().members(key)
// 获取集合中的所有元素
Set<String> members = redisTemplate.opsForSet().members("myset");
  • 从集合中删除一个或多个元素：opsForSet().remove(key, values...)
// 从集合中删除单个元素
redisTemplate.opsForSet().remove("myset", "value1");

// 从集合中删除多个元素
redisTemplate.opsForSet().remove("myset", "value2", "value3");

# ZSet 有序集合 操作

  • 添加元素到有序集合中：opsForZSet().add(key, value, score)
// 添加单个元素到有序集合中，并指定分数
redisTemplate.opsForZSet().add("myzset", "value1", 10.0);

// 添加多个元素到有序集合中，每个元素都有不同的分数
Map<String, Double> values = new HashMap<>();
values.put("value2", 20.0);
values.put("value3", 30.0);
redisTemplate.opsForZSet().add("myzset", values);
  • 获取有序集合的大小：opsForZSet().size(key)
// 获取有序集合的大小
Long size = redisTemplate.opsForZSet().size("myzset");
  • 获取有序集合中某个元素的排名（从小到大）：opsForZSet().rank(key, value)
// 获取指定元素在有序集合中的排名
Long rank = redisTemplate.opsForZSet().rank("myzset", "value1");
  • 获取有序集合中某个元素的排名（从大到小）：opsForZSet().reverseRank(key, value)
// 获取指定元素在有序集合中的排名
Long rank = redisTemplate.opsForZSet().reverseRank("myzset", "value1");
  • 获取有序集合中某个元素的分数：opsForZSet().score(key, value)
// 获取指定元素在有序集合中的分数
Double score = redisTemplate.opsForZSet().score("myzset", "value1");
  • 获取有序集合中指定排名范围内的元素：opsForZSet().range(key, start, end)
// 获取有序集合中排名在 0 到 2 之间的元素（包含两端）
Set<String> values = redisTemplate.opsForZSet().range("myzset", 0, 2);
  • 获取有序集合中指定分数范围内的元素：opsForZSet().rangeByScore(key, min, max)
// 获取有序集合中分数在 10 到 30 之间的元素（包含两端）
Set<String> values = redisTemplate.opsForZSet().rangeByScore("myzset", 10.0, 30.0);
  • 从有序集合中删除一个或多个元素：opsForZSet().remove(key, values...)
// 从有序集合中删除单个元素
redisTemplate.opsForZSet().remove("myzset", "value1");

// 从有序集合中删除多个元素
redisTemplate.opsForZSet().remove("myzset", "value2", "value3");
