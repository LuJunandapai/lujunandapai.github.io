# 07-01 风吹年年

每个人都是25号底片，在路上，在心里 ，在生活中，只是各位还没有发现，有许多人历经艰辛只为找回自己

现状就是 还行
可是今年的小目标感觉完不成...
租房, 小绵羊, 去医院, 电脑坏了...
一天天的 永远不知道明天和意外谁先来
但是啊! 你也并没有那么苦 好像也不够快乐
年年有风, 风吹年年, 慢慢即漫漫....
没有了什么欲望, 只想慢慢过好这一辈子
所愿皆安好, 事事都顺心, 以后慢下来把
