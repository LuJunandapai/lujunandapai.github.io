# 摄影器材

# 焦距 - 透视 - 畸变

# 焦距与视角

超广角 --> 超长焦 (数字依次变大)

# 不同焦距拍摄推荐

  • 超广角 (e.g.16 mm): 大景的风光照片
  • 24 mm: 室内的建筑
  • 35 mm: 人文 人物背景街拍
  • 50 mm: 人像 毕业照
  • 85 mm: 人像特写
  • 长焦 (e.g.200 mm): 人物 建筑 景物的细节
  • 超长焦: 野生动物 较大的月亮

**总结: **

风光建筑: (超)广角+长焦
人像: 35+85+85+135 OR大三元
纪实: 28，35
产品静物: 10OMacro
体育动物: 20O以上

# 曝光 - 补偿

# 感受的判断标准

  • 主体元素正确曝光场景的观感还原
  • 暗部不死黑
  • 亮部不死白
1. 优先保证人物正确曝光,天空轻微过曝是可以接受的

# 曝光三要素

光圈 - 快门 - 感光度

# 光圈 F

  • 光圈为反比关系 F (光圈越大 数值越小 | 光圈越小 数值越大)
  • 大光圈-->背景虚化 小光圈-->远近都清晰

# 快门 S

  • 正比关系 时间越长快门的就越慢 - 时间越短快门就越快

# 感光度 ISO

  • 提高ISO会提高曝光但是降低画质
  • 在昏暗的场景提高ISO会比后期在PS里提高要好很多

# 曝光互易率

  • 光圈 快门 感光度 的每一档的曝光相同
  • 简单来说 增加快门的速度画面变暗 可以提高一档感光度 来保持原有的亮度

# 曝光补偿

  • 白加黑就 调整画面的最终亮度

# 总结

# 曝光组合 - 模式

  • Auto - 全自动: 都不可以调整

  • P - 程序曝光 - 半自动: 只能调节感光度ISO

  • A/AV - 光圈优先: 只不能调节快门

  • S/TV - 快门优先: 只不能调节光圈 (适用于抓拍)

  • M - 全手动挡: 手动调节 (固定场景)

  • B - B门 - 长时间曝光: 按下快门直至松开才会停止曝光

  • C - 自定义: 按照设置保存模板
